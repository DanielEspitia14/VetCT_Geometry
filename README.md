# VetCT Geometry

Software desarrollado en Python para la caracterización geométrica y la estimación volumétrica de regiones de interés en estudios de Tomografía Computarizada (CT), con aplicación en imágenes veterinarias.

## Estado del proyecto

 En desarrollo.

## Características actuales

- Carga de estudios DICOM.
- Navegación entre cortes.
- Visualización de imágenes.
- Segmentación automática del contorno externo.
- Editor de ROI (en desarrollo).

## Tecnologías utilizadas

- Python
- CustomTkinter
- OpenCV
- NumPy
- pydicom
- Pillow
- Matplotlib
